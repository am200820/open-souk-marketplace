# 🤝 المساهمة في المشروع

شكراً لاهتمامك بالمساهمة في Open Souk Marketplace!

## كيفية المساهمة

### 1. Fork المشروع
```bash
git clone https://github.com/am200820/open-souk-marketplace.git
```

### 2. إنشاء فرع feature
```bash
git checkout -b feature/your-feature-name
```

### 3. عمل التغييرات
```bash
git add .
git commit -m "feat: إضافة ميزة جديدة"
```

### 4. Push التغييرات
```bash
git push origin feature/your-feature-name
```

### 5. فتح Pull Request
انقر على "New Pull Request" على GitHub
